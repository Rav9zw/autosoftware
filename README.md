# Instalation Process
1. Clone git@github.com:Rav9zw/autosoftware.git
2. cd .docker
3. docker compose up -d --build
4. Enter docker container
5. Inside container execute points 6 and 7
6. composer install
7. npm install
8. Enter application on [http://localhost/app](http://localhost/app)
